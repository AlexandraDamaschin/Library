First 
enable migrations for a certain context in a certain folder
Enable-Migrations -ContextTypeName Library.Models.ApplicationDbContext -MigrationsDirectory ApplicationUsersMigrations
Enable-Migrations -ContextTypeName Library.Models.LibraryContext -MigrationsDirectory LibraryMigrations

Second 
Add migrations
for the main migration
add-Migration -ConfigurationTypeName Rad301ClubsV1.Migrations.Configuration

for a certain migration
add-migration -ConfigurationTypeName Rad301ClubsV1.Migrations.ClubModelMigrations.Configuration
